CheeseO is a modern professional knowledge sharing platform developed based on Vue 3 + TypeScript. It aims to provide creators with a high-quality content publishing environment while offering learners a wealth of blog posts, video tutorials, and practical tools.

## ✨ Features

### 📝 Content Creation and Sharing
- **Rich Text Editor**: Integrated Tiptap v2, supporting image uploads, code blocks, blockquotes, lists, and various other formats.
- **Multi-type Support**: Support for publishing illustrated blog posts, video courses, and practical tool links.
- **Categories and Tags**: A comprehensive classification system and popular tag cloud for easy content discovery.

### 💰 Points and Withdrawal System
- **Incentive Points**: Users can earn point rewards by publishing high-quality content.
- **Withdrawal Function**: Supports points exchange with a complete bank card withdrawal application process.
- **Transparent Tracking**: Detailed point transaction history and withdrawal status tracking.

### 👥 Social and Interaction
- **Follow System**: Follow your favorite creators to get content updates immediately.
- **Like and Bookmark**: Quickly bookmark content of interest and express support for creators.
- **User Profile**: Display personal works, likes received, follower counts, and other social attributes.

### 🛠 Admin Dashboard
- **Content Review**: A comprehensive pending queue supporting approval/rejection with reason feedback.
- **User Management**: Support for banning/unbanning users who violate platform rules.
- **Category Management**: Flexible management of content categories on the platform.

### 🌐 Internationalization and Responsive Design
- **Full English Interface**: Complete internationalization translation for a professional English user experience.
- **Responsive Layout**: Optimized for mobile, tablet, and desktop devices with mobile gesture support.

## 🚀 Tech Stack

- **Frontend Framework**: Vue 3 (Composition API)
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Headless UI
- **Backend Services**: Firebase (Auth, Firestore, Storage, Hosting)
- **State Management**: Pinia
- **Routing**: Vue Router
- **Editor**: Tiptap v2
- **Icons**: Font Awesome + Iconify

## 📦 Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/YueLi04912/CheeseO.github.io.git
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start development server
   ```bash
   npm run dev
   ```
4. Build for production
   ```bash
   npm run build
   ```

## 📄 Project Structure

- `src/views/user`: User-side views (Home, Blog, Withdrawal, etc.)
- `src/views/admin`: Admin-side views (Content review, User management, etc.)
- `src/components`: Common component library (Editor, Navbar, etc.)
- `src/services`: Firebase database and storage services
- `src/utils`: Data management logic and utility classes
